# Parish Web

Interface web administrativa do sistema Parish desenvolvida com React + Vite.

## 🚀 Tecnologias

- React 18
- TypeScript  
- Vite
- React Router DOM
- Axios

## 📱 Funcionalidades

- ✅ Autenticação (Login)
- ✅ Dashboard administrativo
- ✅ Layout com sidebar
- 🚧 Gestão de Dioceses (em desenvolvimento)
- 🚧 Gestão de Paróquias (em desenvolvimento)
- 🚧 Gestão de Comunidades (em desenvolvimento)

## 🛠️ Como Executar

```bash
npm install
npm run dev
```

Acesse: http://localhost:5173
